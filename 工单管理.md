# 工单管理：

### 1.查询某个仓库的工单

```
GET /api/v1/repos/:owner/:repo/issues
```

参数：

| Name | Type | Desc |
| :--- | :--- | :--- |
| :owner | string | 仓库的所有者，用户或者是组织 |
| :repo | string | 仓库的名字 |

响应：

```
Status: 200 OK
Content-Type: application/json
```

### 2.查询单个工单\(问题\)

```
GET /api/v1/repos/:owner/:repo/issues/:index
```

参数：

| Name | Type | Desc |
| :--- | :--- | :--- |
| :index | int | 工单的id |

响应：

```
Status: 200 OK
Content-Type: application/json
```
