# 工单评论：

### 1.查询某个工单的评论

> 按ID升序排列

```
GET /api/v1/repos/:username/:reponame/issues/:index/comments
```

参数：

| Name | Type | Desc |
| :--- | :--- | :--- |
| since | string | 时间戳格式，\`2006-01-02T15:04:05Z07:00\` |

响应：

```
Status: 200 OK
Content-Type: application/json
```
