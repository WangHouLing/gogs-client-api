# 仓库管理

### 1.创建一个新的仓库

> 给用户或者组织创建一个新的仓库  \`:username\`为仓库的拥有者。

```
POST /admin/users/:username/repos
```

参数：

| Name | Type | Desc |
| :--- | :--- | :--- |
| name | string | 仓库的名字\(必填\) |
| description | string | 描述 |
| private | bool | 私有仓库为\`true\`，公有仓库\`false\`,默认\`false\` |
|  |  |  |
|  |  |  |
|  |  |  |
